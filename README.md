# skills

A collection of reusable skills for AI coding agents (Claude Code, Codex CLI, etc.).

## Skills

| Skill | Description | Source |
|-------|-------------|--------|
| [codex](./codex/) | Run and manage OpenAI Codex CLI sessions for code analysis, refactoring, and automated editing | [skills-directory/skill-codex](https://github.com/skills-directory/skill-codex) |
| [alphaxiv-paper-lookup](./alphaxiv-paper-lookup/) | Look up arxiv papers on alphaxiv.org to get structured AI-generated overviews | [alphaxiv.org](https://www.alphaxiv.org/skills/alphaxiv-paper-lookup/SKILL.md) |
| [gemini](./gemini/) | Run Google Gemini CLI from Claude | Local |
| [julia-dev](./julia-dev/) | General Julia development — Pkg.jl, testing, formatting (BlueStyle), static analysis, package creation, interop | Local |
| [julia-pde](./julia-pde/) | Julia PDE/ODE solving — DifferentialEquations.jl, spatial discretization, sparse linear algebra, AD, HPC parallelism, checkpointing | Local |
| [julia-performance](./julia-performance/) | Julia performance optimization — type stability, allocations, memory layout, profiling, performance annotations | Local (based on [official Julia Performance Tips](https://docs.julialang.org/en/v1/manual/performance-tips/)) |
